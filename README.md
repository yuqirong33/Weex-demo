# 严选 Weex Demo

> further introduction，you can read this [网易严选App感受WEEX开发](https://github.com/zwwill/blog/issues/3)

# Run

## install

```
$ npm install
```

## run web

building web pro

```
$ npm run build 
```

building web pro & running service

```
$ npm run dev & npm run serve 
```

## run ios

install ios platform

``` 
$ weexpack platform add ios
```

build weex bundles

```
$ weex build ios
```
