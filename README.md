# 严选 Weex Demo

# Run

## install

```
$ npm install
```

## run web

building web pro

```
$ npm run build 
```

building web pro & running service

```
$ npm run dev & npm run serve 
```

## run ios

install ios platform

``` 
$ weexpack platform add ios
```

build weex bundles

```
$ weex build ios
```
